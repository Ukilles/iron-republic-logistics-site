# Iron Republic Logistics Website

Deployable Next.js version for **ironrepubliclogistics.com**.

## Included
- App Router Next.js site
- Logo in `public/logo.png`
- Contact info already filled in
- Formspree placeholder for quote requests

## Before you publish
Replace this placeholder in `app/page.tsx`:

```tsx
https://formspree.io/f/YOUR_FORM_ID
```

with your real Formspree endpoint.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Create a GitHub repo and upload these files.
2. Import the repo into Vercel.
3. Deploy.
4. In Vercel, add your domain: `ironrepubliclogistics.com`
5. Copy the DNS records Vercel gives you.
6. Paste those records at your domain registrar.
