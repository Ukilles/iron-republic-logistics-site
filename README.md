# Iron Republic Logistics Website

Complete Next.js website for Iron Republic Logistics.

## Includes
- app/page.tsx
- app/layout.tsx
- app/globals.css
- public/logo.png
- package.json

## Deploy
1. Upload all files to GitHub
2. Import repo into Vercel
3. Deploy
4. Replace `YOUR_FORM_ID` in `app/page.tsx` with your Formspree ID
5. Add your domain in Vercel
